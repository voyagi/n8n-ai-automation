  <action>
Create `workflows/contact-form-ai.json` containing a valid n8n workflow export with these nodes connected in sequence:

**Node 1: Webhook trigger**
- Type: n8n-nodes-base.webhook
- HTTP Method: POST
- Path: `contact-form` (produces URL `/webhook/contact-form`)
- Authentication: Header Auth (credential reference — user will create the credential in n8n UI)
- Respond: "When Last Node Finishes" (respondMode: "lastNode")
- Response Data: First Entry JSON

**Node 2: Set node (Normalize Fields)**
- Type: n8n-nodes-base.set
- Name: "Normalize Fields"
- Mode: Manual Mapping
- Fields to set:
  - `name` → `{{ $json.body.name }}` (String)
  - `email` → `{{ $json.body.email }}` (String)
  - `subject` → `{{ $json.body.subject }}` (String)
  - `message` → `{{ $json.body.message }}` (String)
  - `submittedAt` → `{{ $now.toISO() }}` (String)
- Options: Keep Only Set = true (clean output for downstream nodes)

Note on field paths: n8n Webhook node places POST body fields under `$json.body.*` when content type is JSON. If testing reveals fields are at `$json.*` directly, the expressions will need adjusting — add a comment in the workflow noting this. Research was MEDIUM confidence on the exact path.

**Node 3: IF node (Validate Fields)**
- Type: n8n-nodes-base.if
- Name: "Validate Fields"
- Conditions (ALL must be true):
  - `{{ $json.name }}` is not empty
  - `{{ $json.email }}` is not empty
  - `{{ $json.email }}` matches regex `^[^\s@]+@[^\s@]+\.[^\s@]+$`
  - `{{ $json.subject }}` is not empty
  - `{{ $json.message }}` is not empty

**Node 4 (FALSE branch): Set node (Error Response)**
- Type: n8n-nodes-base.set
- Name: "Error Response"
- Fields:
  - `error` → `"Validation failed"` (String)
  - `message` → `"All fields are required and email must be valid."` (String)

**Node 5 (FALSE branch): Respond to Webhook**

- Type: n8n-nodes-base.respondToWebhook
- Name: "Validation Error"
- Response Code: 400
- Response Body: `{{ $json }}`

**Node 6 (TRUE branch): Set node (Mock AI Response)**
- Type: n8n-nodes-base.set
- Name: "Mock AI Response"
- Purpose: Placeholder that Phase 3 will replace with OpenAI node
- Fields:
  - `category` → `"general"` (String)
  - `summary` → `"Contact form submission received"` (String)
  - `estimatedResponse` → `"within 24 hours"` (String)
  - `name` → `{{ $json.name }}` (String, pass through for downstream)
  - `email` → `{{ $json.email }}` (String, pass through)
  - `subject` → `{{ $json.subject }}` (String, pass through)
  - `message` → `{{ $json.message }}` (String, pass through)
  - `submittedAt` → `{{ $json.submittedAt }}` (String, pass through)
